# HashgraphVisualiser
A visualiser tool for the Hashgraph algorithm displayed by web frontend.

## Usage
#### 1. ssh into AWS
``` ssh -i team9-blockchain.pem ubuntu@18.222.162.68```
#### 2. Run with Docker
```cd  go/src/github.com/group9/HashgraphVisualiser/lachesis```
#### 3. The run
``` docker-compose up```
